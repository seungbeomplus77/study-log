###자바 스크립트 개요

1. 코드를 직접 입력
	- HTML의 <head> 태그나 <body> 태그 안쪽에서 <script> 태그를 이용하여 코드를 입력한다.
		```javascript
		<script type="text/javascript">
		// 자바 스크립트 코드 
		</script> 
		```

2. 자바스크립트 외부 자바스크립트 파일
	- HTML의 <head> 태그나 <body> 태그 안쪽에서 아래와 같이 <script> 태그를 이용하여 외부
	자바스크립트 파일을 추가할 수 있다.
	 ```javascript
	 <script type='text/javascript' src='파일명.js'></script>
	```
